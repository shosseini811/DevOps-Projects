Kubernetes User Management

API Documentation

Welcome to the Kubernetes User Management API. This service provides endpoints for user authentication and management in a Kubernetes cluster.

Authentication Endpoints

POST /api/register
Register a new user with username and password (Admin only)
POST /api/login
Login and receive a JWT token

Admin Endpoints

GET /api/users
List all users (Admin only)

Getting Started

  1. Login to receive your JWT token using the /api/login endpoint
  2. Use the JWT token in the Authorization header for protected endpoints
  3. Admins can create new users using the /api/register endpoint
  4. Admins can view all users using the /api/users endpoint
